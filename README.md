# Rawah's Portfolio

Welcome to my personal portfolio website! This page showcases my projects, skills, and contact information. It’s built using **HTML, CSS, and JavaScript**, with dynamic features such as dark mode and project rendering using JavaScript objects and `sessionStorage`.

## Overview

This portfolio demonstrates my growth as an aspiring interface designer and software developer. Projects are dynamically generated from JavaScript objects and stored in the browser’s `sessionStorage`, allowing easy updates and scalability.

---

## Technologies Used

- **HTML5** – Structure and content  
- **CSS3** – Styling, responsive layout, and dark mode  
- **JavaScript** – Dynamic content rendering, DOM manipulation, and session storage  

Optional libraries/tools:  
- **Figma** – UI/UX prototyping  
- **GitHub** – Version control for projects  

---

## Features

- **Dynamic Projects Section** – Projects are stored as objects in JavaScript and rendered dynamically on page load.  
- **Session Storage** – Project data persists across page reloads.  
- **Dark Mode Toggle** – Allows users to switch between light and dark themes.  
- **Welcome Modal** – Displays a work-in-progress notification when visiting the page.  
- **Skills List** – Dynamically populated list of technical skills.  
- **Featured Content** – Highlights university and personal project links.  
- **Contact Form** – Submits messages with a simulated delay for feedback.
